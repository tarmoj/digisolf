### DigiSolf

 Web based exercises for ear-training (solfeggio). 
 
Realized in the frame of project *"Digitaalse muusikateooria ja solfedžo õppevara loomine"* at [Georg Ots Tallin Music School](www.otsakool.edu.ee)
